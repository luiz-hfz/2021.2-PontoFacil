    1. Introdução

    O Backlog do Produto é um artefato amplo, que comporta as necessidades do produto como um todo. Esse artefato é uma lista que contem todas as funcionalidades desejadas para um determinado produto, ele não precisa estar completo desde o inicio do projeto. Geralmente, é iniciado com aquilo que é mais obvio no primeiro momento e com o passar do tempo ele pode crescer e mudar conforme adquirido s mais conhecimentos acerca do produto e seus usuários.

    2. Épicos

    Épico é uma história de usuário mais longa, geralmente é grande o suficiente para ser dividido em pequenas histórias de usuário.

    3. Histórias de Usuários

    Seria uma descrição detalhada de uma necessidade do usuário do produto sob o ponto de vista desse usuário. Geralmente as histórias de usuário estão mais focadas em “O que” e não no “como”.


    4. Backlog

    Épico US ID História de Usuário
    Épico 01 - Gerenciamento de Registros US01 Eu, como um novo usuário do sistema,poder registrar o ponto de entrada, para que eu saiba quando iniciei o trabalho.
    US02 Eu, como um usuário do sistema, poder registrar o ponto de saída para que eu saiba quando eu saí do trabalho.
    US03 Eu, como um usuário do sistema, desejo registrar o ponto de saída para o almoço para que eu saiba quando eu sai para almoçar.
    US04 Eu, como um usuário do sistema, desejo guardar informações do horário de trabalho para cada dia da semana que eu mantenha os registros organizados e separados.
    US05 Eu, como um usuário do sistema, desejo apagar registros passados para que eu possa manter apenas os registros desejados.
    US06 Eu, como um usuário do sistema, desejo copiar registros passados para que eu possa mover o registro para outro dia.
    US07 Eu, como um usuário do sistema, desejo registrar o ponto de retorno do almoço para que eu saiba quando eu voltei do almoço.
    US08 Eu, como um usuário do sistema, desejo registrar o ponto eletrônico através de uma Tag NFC para que eu tenha mais praticidade e agilidade no registro.
    US09 Eu, como um usuário do sistema, desejo zerar o saldo de horas total para que eu possa recomeçar os registros do zero.
    US10 Eu, como um usuário do sistema, desejo registrar falta para que eu saiba quando eu faltei.
    US11 Eu, como um usuário do sistema, desejo adicionar uma foto do registro do ponto para que tenha certeza que registrei o ponto.
    US12 Eu, como um usuário do sistema, desejo apagar todos os registros para que eu possa apagar tudo que registrei.
    US13 Eu, como um usuário do sistema, desejo registrar folga para que eu saiba quando eu folguei.
    US14 Eu, como um usuário do sistema, desejo editar registros passados que possa corrigir erros que cometi.
    US15 Eu, como um usuário do sistema, desejo registrar férias para que eu saiba quando eu entrei de férias.
    US16 Eu, como um usuário do sistema, desejo escrever uma descrição para cada registro, de forma que possa descrever mais detalhadamente informações sobre o registro.
    US17 Eu, como um usuário do sistema, desejo registrar hora extra para que eu saiba quantas horas extra eu trabalhei.
    Épico 02 - Backup US18 Eu, como usuário do sistema, desejo configurar o horário e o dia do backup,para que possa escolher a hora e o dia mais adequado para backup.
    US19 Eu, como usuário do sistema, desejo armazenar o backup em contas do Google Drive ou Dropbox, para que eu tenha os meus dados salvos em nuvem.
    US20 Eu, como usuário do sistema, desejo fazer backup dos meus registros automaticamente em nuvem ou local, para que eu tenha mais de um tipo de cópia de segurança.
    US21 Eu, como um usuário do sistema, desejo restaurar um backup, para que eu possa resgatar registros anteriormente salvos.
    Épico 03 - Definição de Padrões US22 Eu, como usuário do sistema, desejo configurar os horários de entrada e saída do dia de acordo com registros do mesmo dia , para que eu possa controlar os horários de acordo com a minha rotina.
    US23 Eu, como usuário do sistema, desejo alterar as opções de intervalo, como por exemplo definir um horário de intervalo mínimo, para que eu possa adaptar os registros de acordo com a minha rotina.
    US24 Eu, como usuário do sistema, desejo configurar a tolerância da jornada de trabalho, para que eu tenha controle total sobre as horas trabalhadas.
    US25 Eu, como usuário do sistema, desejo configurar as notificações que o aplicativo informa (aviso de horário de entrada, saída, intervalo), para que eu escolha qual é a notificação mais adequada para mim.
    US26 Eu, como usuário do sistema, desejo configurar a qualidade e resolução da foto armazenada, para que eu possa escolher a qualidade desejada na foto.
    US27 Eu, como usuário do sistema, desejo ativar a notificação de alarme , para que eu seja avisado acerca de alguma atividade que devo realizar
    Épico 04 - Visualização de Registros US28 Eu, como usuario, desejo visualizar inconsistências no registro de ponto, para que eu possa corrigir os erros no registro de ponto.
    US29 Eu, como usuário do sistema, desejo visualizar meu banco de horas no período desejado no formato de gráfico de linhas, para que eu tenha noção do volume de horas trabalhadas.
    US30 Eu, como usuário do sistema, desejo filtrar os parâmetros do gráfico de linhas, para que eu possa inferir informações mais específicas no gráfico.
    US31 Eu, como usuário do sistema, desejo escolher diferentes opções de idiomas, para que eu possa entender as informações apresentadas.
    US32 Eu, como usuário do sistema, desejo visualizar meus registros no dia, na semana e no mês, para que eu acompanhe meu volume de horas trabalhadas no período que desejo.
    US33 Eu, como usuário do sistema, desejo consultar meu saldo de horas do dia, para que eu acompanhe meu volume de horas trabalhadas .
    US34 Eu, como usuário do sistema, desejo gerar um extrato dos meus registros de ponto em qualquer período determinado, para que eu possa apresentar ao meu empregador.
    US35 Eu, como usuário do sistema, desejo consultar o saldo de horas total (do mês), para que eu acompanhe meu volume de horas mensal trabalhadas.
